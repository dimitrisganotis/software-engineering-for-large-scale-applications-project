# Σύστημα Διαχείρισης Συνταγών

Αυτό το project αναπτύχθηκε στα πλαίσια του μεταπτυχιακού μαθήματος "Μηχανική Λογισμικού για Διαδικτυακές & Φορητές Εφαρμογές".

## Οδηγίες Εγκατάστασης και Εκτέλεσης

Η εφαρμογή αποτελείται από 3 μέρη:

1. Βάση Δεδομένων (MySQL)
2. Backend (Java Spring Boot)
3. Frontend (React/TypeScript)

### 1. Βάση Δεδομένων

Για την εκκίνηση της βάσης δεδομένων χρησιμοποιείται Docker. Εκτελέστε την παρακάτω εντολή στον ριζικό φάκελο (`/`):

```bash
docker compose up -d
```

### 2. Backend (Server)

Το backend είναι μια Java Spring Boot εφαρμογή.

- Ανοίξτε το project φάκελο `backend` με το IntelliJ IDEA.
- Εντοπίστε την κλάση `DassApplication.java` (`src/main/java/com/project/dass/DassApplication.java`).
- Κάντε δεξί κλικ και **Run 'DassApplication'**.

Ο server θα ξεκινήσει στο `http://localhost:8080`.

### 3. Frontend (Client)

Το frontend είναι μια React εφαρμογή.

- Ανοίξτε ένα τερματικό και μεταβείτε στον φάκελο `frontend`:
  ```bash
  cd frontend
  ```
- Εγκαταστήστε τα dependencies:
  ```bash
  npm install
  ```
- Εκκινήστε την εφαρμογή:
  ```bash
  npm run dev
  ```

### Πρόσβαση στην Εφαρμογή

Ανοίξτε τον browser στη διεύθυνση: **http://localhost:5173**
